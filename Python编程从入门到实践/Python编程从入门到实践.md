## 列表
> 列表由一系列按特定顺序排列的元素组成
```py {.line-numbers}
bicycles = ['trek', 'cannondale', 'redline', 'specialized']
print(bicycles)
print(bicycles[0])
print(bicycles[0].title())
# 输出最后一个元素
print(bicycles[-1])
print(bicycles[-2])
```
